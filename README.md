# Python-Programming

1.	Introduction to Python Programming
a.Demonstration of installation and configuration of Anaconda and Spyder IDE.
b.Write a Python program to accept user input and display the entered value.
c.Introduction to Git and GitHub and creation of repositories with basic version control operations.
d.Introduction to the LeetCode online coding challenge platform for developing problem-solving skills.
		
2.	Control Structures
a.Write a Python program to check whether a given year is a leap year.
b.Write a Python program to find the factorial of a number using loops.
Application-oriented and practical scenario–based programming problems are included.
			
3.	Nested Loops, Functions, and Recursion
a.Write a Python program using nested loops to print the following pattern for n rows:

are included.
